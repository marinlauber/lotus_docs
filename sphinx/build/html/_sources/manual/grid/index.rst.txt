######
 Grid
######

   Generally speaking, the geometry in Lotus normally is placed in the origin of coordinates. With the flow being initiated positive in X.

   The main function to control the size of the grid is the function stretch.

   There are a number of diferences when setting up the grid for two, three dimensional; parallel and serial computation simlations.
