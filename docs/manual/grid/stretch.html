Stretch¶

stretch(a, n, dom_neg, uni_neg, uni_pos, dom_pos, h_max, h_min, prnt)
This function determines the uniform and exponential spacing of a 1D grid
Parameters

  • n (int) – number of points

  • dom_neg (real) – distance to negative domain boundary from origin

  • uni_neg (real) – negative extent of uniform section from origin

  • uni_pos (real) – positive extent of uniform section from origin

  • dom_pos (real) – distance to negative domain boundary from origin

  • h_max (Optional[real]) – distance to negative domain boundary from origin

  • h_min (Optional[real]) – distance to negative domain boundary from origin

Return type

line
